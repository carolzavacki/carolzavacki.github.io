Projects

Calculator

A functional math calculator that I've built with Javascript. It can do the 4 simple math operations : addition, substraction, multiplication and division. The app allows the user to do math with decimals and also counts for the times the user may introduce 2 decimal points by error. The app listens to click only events (so it does not function with keyboard numeric input), but I plan to recreate this app as soon as I'll learn a Javascript framework.

Variety of foods

This app represents a plain one page presentation website. The main technology used here was CSS, with barely any Javascript. All the icons used are Font Awesome Icons, taken from fontawesome.com, along with their specific CSS. The app also has a dropdown menu, along with icons, cards and gallery that tries to represent a generic food restaurant.

Infinite scroll

This app generates an infinite scroll of pictures by using the Unsplash API and the scroll event listener. It's main focus was working around an API to better understand them. The app displays a loader SVG from loading.io when you first open it, and it requests photos from Unsplash as you scroll down. By hovering over a photo it displays it's name, and by clicking on any image it will send you to the Unsplash website on that specific picture.
